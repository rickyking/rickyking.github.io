Collaborative Filtering for Implicit Feedback on Browsing Hits Data

Posted on 2018-03-17 Edited on 2021-03-01 In Notes Disqus:

Recently, I worked on recommendation system in SSENSE. I have employed the collaborative filtering algorithm based on implicit feedback, which is the number of hits per item.

The methodology part is based on:

Hu, Yifan, Yehuda Koren, and Chris Volinsky. “Collaborative filtering for implicit feedback datasets.” Data Mining, 2008. ICDM’08. Eighth IEEE International Conference on. Ieee, 2008.

While the implicit feedback is the number of hits on the product/item page.

Methodology

Setting

u - user, v another user
i - item, j another item

ruir_{ui}rui​ - observations, e.g. hits, time spent, rui=0r_{ui}=0rui​=0 means missing observation

Preference:

Pui={1,rui>00,rui=0P_{ui} = \begin{cases} 1, & r_{ui} > 0 \\ 0, & r_{ui} = 0 \end{cases} Pui​={1,0,​rui​>0rui​=0​

Confidence:

Cui={1+α×rui,rui>01,rui=0C_{ui} = \begin{cases} 1 + \alpha \times r_{ui}, & r_{ui} > 0 \\ 1, & r_{ui} = 0 \end{cases} Cui​={1+α×rui​,1,​rui​>0rui​=0​

Goal

Find a vector xu∈Rfx_u \in \mathbb{R}^fxu​∈Rf for each user u, and a vector yi∈Rfy_i \in \mathbb{R}^fyi​∈Rf for each item i, thus pui=xuTyip_{ui} = x_u^T y_ipui​=xuT​yi​

Math to English

The vectors strive to map users and items into a common latent factor space where they can be directly compared.

Cost Function

\min_{x_\*, y_\*} \sum_{u,i} c_{ui}(p_{ui}-x^T_uy_i)^2 + \lambda(\sum_{u} \| x_u \| ^ 2 + \sum_{i} \| y_i \| ^2)

Algorithm: alternating-least-squares (als)

alternate between recomputing user-factors and item-factors.

Step 1. recomputing all user factors XXX

xu=(YTCuY+λI)−1YTCup(u)where YTCuY=YTY+YT(Cu−I)Y x_u = (Y^T C^u Y + \lambda I)^{-1} Y^T C^u p(u) \\\\ where\ Y^T C^u Y = Y^T Y + Y^T (C^u - I)Y xu​=(YTCuY+λI)−1YTCup(u)where YTCuY=YTY+YT(Cu−I)Y

Step 2. recomputing all item factors YYY

yi=(XTCiX+λI)−1XTCip(i)y_i = (X^T C^i X + \lambda I)^{-1} X^T C^i p(i) yi​=(XTCiX+λI)−1XTCip(i)

Step 3. Iterate over step 1 and step 2, till stablize

Explaining Recommendation

As the preference for item i given user u is

pui^=yiTxu=yiT(YTCuY+λI)−1YTCup(u).\hat{p_{ui}} = y^T_i x_u = y_i^T (Y^T C^u Y + \lambda I)^{-1} Y^T C^u p(u). pui​^​=yiT​xu​=yiT​(YTCuY+λI)−1YTCup(u).

Now, we denote Wu=(YTCuY+λI)−1W^u = (Y^T C^u Y + \lambda I)^{-1}Wu=(YTCuY+λI)−1 as weighting matrix associated with user u. Accordingly, the weighted similarity between items i and j from u's viewpoint is denoted by siju=yiTWuyjs^u_{ij} = y_i^T W^u y_jsiju​=yiT​Wuyj​.
